# Manish Yadav Academic Website

Simple GitHub Pages–ready academic website.

## Files
- `index.html` — main page
- `style.css` — styling
- `cv.pdf` — your CV download
- `assets/` — badge-style SVGs for affiliations

## Before publishing
Replace these placeholder links in `index.html`:
- `Google Scholar`
- `GitHub`

The LinkedIn link is already wired to:
- `https://www.linkedin.com/in/manish-yadav-191370a2/`

## Publish with GitHub Pages
1. Create a new repository, ideally `manishyadav.github.io`
2. Upload all files from this folder
3. Commit to the default branch
4. In GitHub Settings → Pages, set the source to `Deploy from a branch`
5. Select the main branch and `/root`
6. Your site will appear at `https://manishyadav.github.io/`

## Notes
The badges are simple original SVG designs using text labels, not official logo files.
